## Tech Stack

Project ini dibangun menggunakan:

* **React 19** → Library UI
* **Vite** → Build tool super cepat
* **React Router DOM** → Routing halaman
* **Tailwind CSS** → Styling modern & utility-first
* **Axios** → HTTP client untuk API
* **ESLint** → Linting & code quality

## Prerequisites

Pastikan sudah install:

* Node.js (disarankan v18+)
* npm / yarn / pnpm

---

## Setup Project

### 1. Clone Repository

```bash
git clone https://github.com/481n4y4/cathering-mama-pasha.git
cd cathering-mama-pasha
```

---

### 2. Install Dependencies

```bash
npm install
```

---

### 3. Jalankan Development Server

```bash
npm run dev
```

---

### 4. Build Production

```bash
npm run build
```

---

### 5. Preview Build

```bash
npm run preview
```
